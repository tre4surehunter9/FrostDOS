# FrostDOS

## A OS (more of a kernel) coded entirely in Rust

![Screenshot](Screenshot_20260601_135311.png)

## Features
* Basic Shell
* Heap Allocation
* Keyboard Input

## Commands
* echo - Echos back your Message
* about - Kernel info
* clear - Clear the screen
* help - Show the help message
* reboot - Reboots the system
